# security-log-monitor# Automated Security Log Monitoring and Alert System

## Overview
This project demonstrates how Python can be used to automate the monitoring of security log files. The script scans log entries and generates alerts when suspicious activity is detected.

## Features
- Reads security log files
- Detects failed login attempts
- Detects unauthorized access attempts
- Detects access denied events
- Generates alerts for suspicious activity

## Requirements
- Python 3

## Files
- main.py
- sample_log.txt

## How to Run

python main.py

## Example Output

ALERT: Failed login attempt from 192.168.1.100
ALERT: Access denied for guest account
ALERT: Unauthorized access attempt detected
